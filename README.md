# Intent Classification - Text Mining Project

This project studies intent classification on banking customer queries.

The objective is to compare classical text mining approaches such as Bag of Words and TF-IDF with semantic representations such as sentence embeddings.

## Structure

- `data/raw/` : raw dataset files
- `data/processed/` : cleaned dataset
- `scripts/` : Python scripts
- `figures/` : generated figures
- `results/` : metrics and reports
- `reports/` : presentation and report materials
- `models/` : trained models
